AI Agent Platform Timeline

May 16, 2025

Strands SDK Launched

AWS announces Strands Agents, an open source SDK for building AI agents with a model-driven approach. The framework enables developers to build agents by defining prompts and tool lists with minimal code, scaling from simple to complex use cases.

July 16, 2025

Amazon Bedrock AgentCore Preview

AWS introduces AgentCore in preview, a comprehensive set of enterprise-grade services including Runtime, Memory, Observability, and Identity to help developers deploy and operate AI agents at scale using any framework and model.

July 14, 2025

Kiro Public Preview

Amazon launches Kiro in public preview, an agentic IDE that introduces spec-driven development to transform "vibe coding" into production-ready software. Built on VS Code OSS and powered by Anthropic's Claude models, Kiro enforces structured workflows from requirements through deployment.

October 13, 2025

Amazon Bedrock AgentCore Generally Available

AWS announces General Availability of AgentCore with enhanced features including VPC support, AWS PrivateLink, CloudFormation integration, and Agent-to-Agent (A2A) protocol support for production-ready multi-agent orchestration.

November 17, 2025

Kiro Generally Available

Kiro reaches General Availability with full production support, offering developers powerful AI-assisted coding capabilities with enhanced security and performance features across macOS, Windows, and Linux.
